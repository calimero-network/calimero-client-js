Calimero Client JS

TypeScript SDK for interacting with Calimero nodes — RpcClient for queries and mutations, SubscriptionsClient for real-time WebSocket and SSE event streams.

Start Here

1

Install

npm install @calimero-network/calimero-client

Or: yarn add / pnpm add

2

Create RpcClient

import { RpcClient } from '@calimero-network/calimero-client';

const client = new RpcClient(nodeUrl, applicationId);
3

Query or Mutate

const result = await client.query({
  method: 'get_value',
  args: { key: 'foo' }
});
