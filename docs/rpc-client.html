RPC Client

Execute WASM application methods — query and mutate

Setup

import { RpcClient, WsSubscriptionsClient } from '@calimero-network/calimero-client';

const rpc = new RpcClient({
  nodeUrl: 'http://localhost:2428',
  contextId: 'ctx_abc...',
});

// Set access token (obtained from auth flow):
rpc.setToken(accessToken);

Methods

rpc.query(method, args)
Execute a read-only WASM method. Returns the JSON-parsed response body. args: plain JS object.
rpc.mutate(method, args)
Execute a state-mutating WASM method. Same signature as query.
rpc.setToken(token)
Set or update the JWT access token used for authentication.
rpc.setContextId(id)
Change the active context. Subsequent calls use the new context ID.

Calling WASM Methods

// Query — read-only
const posts = await rpc.query('get_posts', {});
console.log(posts);

// Mutate — state change
await rpc.mutate('create_post', {
  title: 'Hello World',
  body: 'Content here',
});

// With typed return value
const post = await rpc.query<Post>('get_post', { id: 42 });

Error Handling

try {
  await rpc.mutate('create_post', { title: '' });
} catch (err) {
  if (err.status === 401) {
    // Token expired — refresh and retry
    const newToken = await refreshToken();
    rpc.setToken(newToken);
    await rpc.mutate('create_post', { title: '' });
  } else if (err.status === 400) {
    console.error('Bad request:', err.body);
  }
}
