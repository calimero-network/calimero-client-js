Auth API

Login, token management, and client key generation — accessed via apiClient.auth()

Login & Redirect

login() redirects the browser to the Calimero auth service. After the user authenticates, the service redirects back to your callbackUrl with access_token and refresh_token in the URL.

login(request)
Redirect to the auth service login page. Supports mode, manifestUrl (package-based), and legacy applicationId.
import { apiClient } from '@calimero-network/calimero-client';

// Package-based (recommended)
apiClient.auth().login({
  url: 'https://your-node.calimero.network',
  callbackUrl: window.location.href,
  permissions: ['read', 'write'],
  mode: 'multi-context',
  manifestUrl: 'https://registry.calimero.network/my-app/1.0.0',
});

// Legacy (hash-based app ID)
apiClient.auth().login({
  url: 'https://your-node.calimero.network',
  callbackUrl: window.location.href,
  permissions: ['read'],
  applicationId: 'app_hash_here',
  applicationPath: '/app',
});

Token Management

refreshToken(request)
Exchange an expired access token for a new pair. Pass both access_token and refresh_token.
requestToken(requestBody)
Low-level token request with full control over auth method, public key, and provider data.
const tokens = await apiClient.auth().refreshToken({
  access_token: currentAccessToken,
  refresh_token: currentRefreshToken,
});

if (!tokens.error) {
  setAccessToken(tokens.data.access_token);
  setRefreshToken(tokens.data.refresh_token);
}

Challenge-Based Auth

Used to authenticate with NEAR or other signature-based providers. The flow: request a challenge → sign it → submit the signed message to requestToken.

getChallenge()
Get a server-issued challenge nonce to sign. Returns { challenge, nonce }.
getProviders()
List configured auth providers (NEAR, MetaMask, etc.) and their configuration status.
// 1. Get challenge
const ch = await apiClient.auth().getChallenge();
const { challenge, nonce } = ch.data!;

// 2. Sign the challenge with NEAR wallet (application-specific)
const signed = await wallet.signMessage({ message: challenge, nonce: Buffer.from(nonce, 'base64') });

// 3. Exchange for JWT tokens
const token = await apiClient.auth().requestToken({
  auth_method: 'near',
  public_key: signed.publicKey,
  client_name: 'my-app',
  timestamp: Date.now(),
  provider_data: { accountId: signed.accountId, signature: signed.signature },
});

Client Keys

generateClientKey(request)
Generate a scoped client key for a specific context and identity. Returns a JWT pair tied to that context.
const key = await apiClient.auth().generateClientKey({
  context_id: 'ctx_abc...',
  context_identity: 'pub_key_hex',
  permissions: ['read', 'write'],
  target_node_url: 'https://your-node.calimero.network',
});
// key.data: { access_token, refresh_token }

Types

interface LoginRequest {
  url: string;
  callbackUrl: string;
  permissions: string[];
  mode?: string;            // 'single-context' | 'multi-context' | 'admin'
  manifestUrl?: string;     // package-based app resolution
  applicationId?: string;   // legacy hash-based ID
  applicationPath?: string;
}

interface Provider {
  name: string;
  type: string;
  description: string;
  configured: boolean;
}
