Blob API

Binary file upload, download, and metadata — accessed via apiClient.blob()

Upload

uploadBlob(file, onProgress?, expectedHash?)
Upload a File object via HTTP PUT. Optional onProgress callback receives a 0–100 number. Optional expectedHash for server-side integrity verification. Returns { blobId, size }.
import { apiClient } from '@calimero-network/calimero-client';

// From a file input element
const file: File = input.files[0];

const res = await apiClient.blob().uploadBlob(
  file,
  (pct) => console.log(`Upload: ${pct}%`),
);

if (!res.error) {
  console.log('Blob ID:', res.data.blobId);
  console.log('Size:', res.data.size, 'bytes');
}

Download

downloadBlob(blobId, contextId)
Download a blob as a browser Blob object. Throws on error (unlike other methods which return an error object).
// Download and trigger browser save dialog
const blob = await apiClient.blob().downloadBlob(blobId, contextId);

const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = 'file';
a.click();
URL.revokeObjectURL(url);

List, Metadata & Delete

listBlobs()
List all blobs stored on the node. Returns { blobs: { blobId, size }[] }.
getBlobMetadata(blobId)
Fetch metadata via HTTP HEAD. Returns { blobId, size, fileType } extracted from response headers.
deleteBlob(blobId)
Permanently delete a blob by ID.
// List all blobs
const list = await apiClient.blob().listBlobs();
list.data?.blobs.forEach((b) => console.log(b.blobId, b.size));

// Check metadata without downloading
const meta = await apiClient.blob().getBlobMetadata(blobId);
console.log(meta.data?.fileType, meta.data?.size);

// Delete
await apiClient.blob().deleteBlob(blobId);

Types

interface BlobUploadResponse {
  blobId: string;
  size: number;
}

interface BlobMetadataResponse {
  blobId: string;
  size: number;
  fileType: string;  // MIME type from Content-Type or x-blob-mime-type header
}

interface BlobInfo {
  blobId: string;
  size: number;
}

interface BlobApi {
  uploadBlob(file: File, onProgress?: (n: number) => void, expectedHash?: string): ApiResponse<BlobUploadResponse>;
  downloadBlob(blobId: string, contextId: string): Promise<Blob>;
  getBlobMetadata(blobId: string): ApiResponse<BlobMetadataResponse>;
  listBlobs(): ApiResponse<BlobListResponseData>;
  deleteBlob(blobId: string): ApiResponse<void>;
}

Note: downloadBlob throws on error instead of returning an error object. Wrap in try/catch.
