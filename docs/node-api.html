Node API

Contexts, identities, applications, capabilities, and group management — accessed via apiClient.node()

Contexts

health(request)
Check node health. request.url overrides the stored endpoint.
getContexts()
List all contexts on the node. Returns { contexts: Context[] }.
getContext(contextId)
Fetch a single context by ID.
createContext(applicationId, jsonParams)
Create a new context for the given application. Pass '' for no init params.
deleteContext(contextId)
Delete a context. Returns { isDeleted: boolean }.
fetchContextIdentities(contextId)
List identities owned by the caller inside a context.
import { apiClient } from '@calimero-network/calimero-client';

// List contexts
const res = await apiClient.node().getContexts();
if (!res.error) console.log(res.data.contexts);

// Create a context
const ctx = await apiClient.node().createContext('app_id', '');
console.log(ctx.data?.contextId);

Identity

createNewIdentity()
Generate a new context identity (key pair). Returns { publicKey, privateKey }.
checkAuth()
Verify the current JWT is valid and return its decoded payload.
const identity = await apiClient.node().createNewIdentity();
// { publicKey: '...', privateKey: '...' }

const auth = await apiClient.node().checkAuth();
// { payload: { context_id, executor_public_key, ... } }

Invite & Join

contextInvite(contextId, inviterId, validForSeconds)
Create a signed invitation for a new member. Returns a SignedOpenInvitation.
joinContext(invitation, newMemberPublicKey)
Join an existing context using a SignedOpenInvitation.
// Inviter side — create invitation valid for 5 minutes
const inv = await apiClient.node().contextInvite(contextId, inviterId, 300);
const payload = inv.data?.data; // SignedOpenInvitation

// Joiner side — pass invitation + new member's public key
const join = await apiClient.node().joinContext(payload, newMemberPublicKey);
console.log(join.data?.contextId);

Applications

getInstalledApplications()
List all applications installed on the node.
getInstalledApplicationDetails(appId)
Fetch details for a specific installed application.
installApplication(url, metadata?, hash?)
Install a WASM application from a URL. Returns { applicationId }.
uninstallApplication(applicationId)
Remove an installed application from the node.
const apps = await apiClient.node().getInstalledApplications();
apps.data?.apps.forEach((app) => console.log(app.id, app.version));

// Install from URL
const installed = await apiClient.node().installApplication('https://example.com/app.wasm');
console.log(installed.data?.applicationId);

Context Details

getContextClientKeys(contextId)
List client keys associated with a context.
getContextUsers(contextId)
List member identities inside a context.
getContextStorageUsage(contextId)
Get storage consumption for a context in bytes.

Capabilities

Capabilities control what operations context members can perform. Pass an array of [ContextIdentity, Capability] tuples.

grantCapabilities(contextId, request)
Grant capabilities to one or more context identities.
revokeCapabilities(contextId, request)
Revoke capabilities from one or more context identities.
await apiClient.node().grantCapabilities(contextId, {
  capabilities: [['identity_abc', 'write']],
  signer_id: 'signer_pub_key',
});

Group Management

Groups are multi-context collections with role-based membership (Admin, Member, ReadOnly).

Group lifecycle

listGroups()
Return all groups visible to the current identity.
createGroup(request)
Create a group. Accepts applicationId, optional parentGroupId, upgradePolicy, and alias.
getGroupInfo(groupId)
Fetch metadata for a group (member count, context count, alias, etc.).
deleteGroup(groupId)
Delete a group and all its state.

Members

listGroupMembers(groupId)
List members with their roles. Also returns selfIdentity.
addGroupMembers(groupId, request)
Add members with specified roles. request.members: Array<{ identity, role }>.
removeGroupMembers(groupId, request)
Remove members by identity. request.members: string[].
setMemberCapabilities(groupId, memberId, capabilities)
Set capability bitmask for a group member.
getMemberCapabilities(groupId, memberId)
Get capability bitmask for a group member.

Contexts inside groups

listGroupContexts(groupId)
List contexts belonging to a group.
joinGroupContext(groupId, contextId)
Join an existing context within a group.

Invite & join groups

createGroupInvitation(groupId)
Generate a SignedOpenInvitation to share with prospective members.
joinGroup(request)
Join a group using a SignedOpenInvitation. Optional groupAlias.
// Create a group
const g = await apiClient.node().createGroup({
  applicationId: 'app_id',
  alias: 'my-group',
});
const groupId = g.data?.data.groupId;

// Add members
await apiClient.node().addGroupMembers(groupId, {
  members: [{ identity: 'pub_key', role: 'Member' }],
});

// Invite someone to join
const inv = await apiClient.node().createGroupInvitation(groupId);
// Send inv.data.data (SignedOpenInvitation) to the new member

// Joiner side
await apiClient.node().joinGroup({ invitation: signedInv, groupAlias: 'my-group' });

Types

type GroupMemberRole = 'Admin' | 'Member' | 'ReadOnly';

interface GroupInfo {
  groupId: string;
  appKey: string;
  targetApplicationId: string;
  upgradePolicy: string;
  memberCount: number;
  contextCount: number;
  alias?: string;
}

interface GroupMember {
  identity: string;
  role: GroupMemberRole;
  alias?: string;
}

interface SignedOpenInvitation {
  invitation: InvitationFromMember;
  inviterSignature: string;
  applicationId?: number[];
  blobId?: number[];
  source?: string;
  groupId?: number[];
}
