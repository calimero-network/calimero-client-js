Admin API

Key management, permissions, package registry, and context queries — accessed via apiClient.admin()

Key Management

Two-tier key model: root keys are identity anchors (one per auth method), client keys are scoped tokens derived from root keys.

getRootKeys()
List all root keys on the node.
getClientKeys()
List all client keys across all root keys.
addRootKey(rootKeyRequest)
Register a new root key. Provide public_key, auth_method, and provider_data.
revokeRootKey(keyId)
Revoke a root key and all client keys derived from it.
revokeClientKey(rootKeyId, clientId)
Revoke a specific client key without affecting other keys.
import { apiClient } from '@calimero-network/calimero-client';

const rootKeys = await apiClient.admin().getRootKeys();
rootKeys.data?.forEach((k) => console.log(k.key_id, k.auth_method));

// Revoke a root key
await apiClient.admin().revokeRootKey('key_id_here');

Permissions

setKeyPermissions(keyId, request)
Add or remove permissions on a client key. request.add and request.remove are both optional string[].
await apiClient.admin().setKeyPermissions('client_key_id', {
  add: ['write', 'admin'],
  remove: ['read_only'],
});

Package Management

Query the Calimero package registry and install applications from it.

getPackages()
List all packages available in the registry. Returns { packages: string[] }.
getPackageVersions(packageName)
List available versions for a package. Returns { versions: string[] }.
getPackageLatest(packageName)
Get the applicationId of the latest installed version of a package.
installApplication(request)
Install an application from the registry. Provide url, package, version, and metadata.
const pkgs = await apiClient.admin().getPackages();
console.log(pkgs.data?.packages);

const versions = await apiClient.admin().getPackageVersions('network.calimero.meropass');
console.log(versions.data?.versions);

// Install a specific version
const result = await apiClient.admin().installApplication({
  url: 'https://registry.calimero.network',
  package: 'network.calimero.meropass',
  version: '1.0.0',
  metadata: [],
});
console.log(result.data?.data.applicationId);

Context Queries

getContextsForApplication(applicationId)
List all contexts running a given application.
getContextsWithExecutorsForApplication(applicationId)
Same as above but also includes the list of executor identities per context.
const ctxs = await apiClient.admin().getContextsForApplication('app_id');
ctxs.data?.data.contexts.forEach((c) => console.log(c.id));

const withExec = await apiClient.admin().getContextsWithExecutorsForApplication('app_id');
withExec.data?.contexts.forEach((c) => console.log(c.id, c.executors));

Types

interface RootKey {
  key_id: string;
  public_key: string;
  auth_method: string;
  created_at: number;
  revoked_at: number;
}

interface ClientKey {
  client_id: string;
  root_key_id: string;
  name: string;
  permissions: string[];
  created_at: number;
  revoked_at?: number;
  is_valid: boolean;
}
